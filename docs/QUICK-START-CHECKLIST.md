# GrantIQ Sales System: Quick-Start Checklist

**Print this out. Check boxes as you complete tasks.**
**Target completion: 14 days**

---

## DAY 1: EXECUTIVE APPROVAL

- [ ] CEO reads SALES-SYSTEM-EXECUTIVE-SUMMARY.md (15 min)
- [ ] Leadership approves budget: $60K-$80K (first 90 days)
- [ ] Leadership approves hiring: 1-2 sales people + 1 CRM admin
- [ ] Sales Manager assigned (owns execution)
- [ ] Meeting scheduled: Brief team on sales initiative (30 min)

**Owner:** CEO / Sales Manager

---

## DAY 2-3: HIRING & TOOL SETUP

### Hiring
- [ ] Job description posted for Sales Rep (if hiring new)
- [ ] Internal candidate identified (if promoting)
- [ ] CRM Admin assigned or hired
- [ ] Interviews scheduled (if hiring)

**Owner:** HR + Sales Manager

### Tool Purchases
- [ ] HubSpot Sales Hub account created (3 users: Admin, Sales Rep, Manager)
  - Alternative: Pipedrive if preferred
  - Cost: ~$180/user/month
- [ ] Email platform confirmed (Gmail or Office 365 with tracking)
- [ ] Calendar software confirmed (Calendly or native CRM)
- [ ] Slack workspace ready (for notifications)

**Owner:** CRM Admin

---

## DAY 4-5: CRM SETUP

### Configuration
- [ ] CRM Admin reads crm-automation-configuration.md (Part 1: Field Setup) (2 hours)
- [ ] All custom fields created in CRM:
  - [ ] Basic fields (Name, Email, Phone, Title, Organization)
  - [ ] Firmographic fields (Size, Budget, Sector, Organization Type)
  - [ ] Qualification fields (A-F scores)
  - [ ] Engagement fields (Email opens, clicks, replies)
  - [ ] Sales pipeline fields (Stage, Deal value, Close date)
- [ ] 7-stage pipeline created:
  - [ ] PROSPECT (0% probability)
  - [ ] QUALIFIED (30% probability)
  - [ ] CONSULTATION_SCHEDULED (50% probability)
  - [ ] CONSULTATION_COMPLETED (60% probability)
  - [ ] PROPOSAL_SENT (75% probability)
  - [ ] NURTURE (10% probability)
  - [ ] WON (100% probability) / LOST (0% probability)

**Owner:** CRM Admin

### Email Template Setup
- [ ] 4 cold email sequences created as templates:
  - [ ] Sequence A Email 1: Problem Identification (use template from sales-automation-system.md Section 1)
  - [ ] Sequence A Email 2: Case Study + Social Proof
  - [ ] Sequence A Email 3: Scarcity + Close
  - [ ] Sequence B Email 1: Risk/Compliance Angle (for mid-market)
  - [ ] Sequence B Email 2: ROI Framework
  - [ ] Sequence B Email 3: Close/Calendar
  - [ ] Sequence C Email 1: Usage-Based (SaaS users)
  - [ ] Sequence C Email 2: Objection Prevention
  - [ ] Sequence C Email 3: Time-Limited Offer
  - [ ] Sequence D Email 1: Win-Back (Dormant accounts)
  - [ ] Sequence D Email 2: Offer + Discount
- [ ] Discovery call confirmation email template
- [ ] Post-call follow-up email template
- [ ] Proposal cover email template
- [ ] Proposal follow-up email templates (x3)

**Owner:** Sales Manager + CRM Admin

---

## DAY 6-7: AUTOMATION WORKFLOWS

- [ ] CRM Admin reads crm-automation-configuration.md (Part 2: Automation Workflows) (2 hours)
- [ ] Build 6 automation workflows in CRM:
  - [ ] Workflow 1: Cold Email Sequence Automation (Email 1 → wait → Email 2 → wait → Email 3)
  - [ ] Workflow 2: Discovery Call Automation (confirmation → 1-day reminder → pre-call email → post-call task)
  - [ ] Workflow 3: Lead Scoring (auto-calculate A-F scores, update total, assign to segment)
  - [ ] Workflow 4: Proposal Follow-up (send proposal → wait 2 days → send reminder → wait 2 days → send 2nd reminder → task to call)
  - [ ] Workflow 5: Nurture Drip (assign to nurture sequence based on disqualification reason, monthly emails)
  - [ ] Workflow 6: SaaS User Conversion (when user signs up → send welcome → monitor logins → send coaching emails based on engagement)
- [ ] All workflows tested with sample contacts
- [ ] Workflows activated (live)

**Owner:** CRM Admin

### Dashboard Setup
- [ ] Weekly sales dashboard created:
  - [ ] Pipeline by stage (# of deals, $ value)
  - [ ] This week's activities (emails sent, calls held, consultations scheduled)
  - [ ] Email metrics (open rate, reply rate, click rate)
  - [ ] Channel performance (Cold Email, SaaS, LinkedIn, Referrals)
- [ ] Pipeline report created (for weekly reporting)
- [ ] Lead scoring report created (shows distribution of scores)

**Owner:** CRM Admin

---

## DAY 8: LEAD LIST ACQUISITION

- [ ] Prospect list sources identified:
  - [ ] Nonprofit sector list (DonorTrends, Foundation Center, GiveWell)
    - Target: 1,000+ nonprofits with $500K+ budgets
  - [ ] Small business list (LinkedIn Sales Navigator, Apollo, Hunter)
    - Target: 500+ businesses in eligible sectors
  - [ ] SaaS free user list (export from app)
    - Target: High-engagement users (3+ logins, active past 30 days)
  - [ ] LinkedIn prospect list (manual build or Sales Navigator)
    - Target: 500+ prospects by title + sector
- [ ] Lists acquired or being acquired
- [ ] Total target: 2,000+ qualified prospects for Phase 1

**Owner:** Sales Manager + Ops

---

## DAY 9: DATA CLEANUP

- [ ] Lists imported into CRM (or staged for import)
- [ ] Duplicates removed:
  - [ ] Automated deduplication (CRM can do this)
  - [ ] Manual spot-checks completed
- [ ] Email validation:
  - [ ] ZeroBounce or similar email validator run
  - [ ] Invalid emails flagged or removed
- [ ] Data enrichment (if using a tool like Apollo or Hunter):
  - [ ] Company size populated
  - [ ] Industry/Sector populated
  - [ ] Annual budget estimated
- [ ] Leads tagged by source:
  - [ ] COLD_EMAIL_SOURCE_A
  - [ ] COLD_EMAIL_SOURCE_B
  - [ ] SAAS_USER
  - [ ] LINKEDIN
  - [ ] REFERRAL
- [ ] Data import complete, ready for launch

**Owner:** CRM Admin + Sales Manager

---

## DAY 10: TEAM TRAINING (HALF DAY)

### Sales Manager Preparation
- [ ] Sales Manager reads sales-automation-system.md (Full read, 2 hours)
- [ ] Sales Manager reads sales-operational-playbook.md (Full read, 2 hours)
- [ ] Sales Manager prepares discovery call script (print from playbook)
- [ ] Sales Manager prepares objection handling cards (print 5 cards, laminate or save as reference)

### Sales Team Training
- [ ] Sales Rep(s) attend 2-hour training session:
  - [ ] Review: Cold email sequences (why they work, how to personalize)
  - [ ] Review: Discovery call script (walk through line by line)
  - [ ] Review: A-Z qualification framework (score each criterion)
  - [ ] Review: CRM workflows (how emails auto-send, how tasks auto-assign)
  - [ ] Review: Daily/weekly process (what they do each day)
- [ ] Sales Rep reads sales-operational-playbook.md (1 hour)
- [ ] Mock discovery calls held (30 min, role-play with manager)
  - [ ] Sales Rep plays prospect, Manager plays sales person
  - [ ] Switch roles and repeat
  - [ ] Manager gives feedback on script adherence + closing

**Owner:** Sales Manager

---

## DAY 11: LAUNCH PREP

### Email Personalization
- [ ] Sales Manager drafts 10 sample cold emails (Email 1, Sequence A):
  - [ ] Each email is hand-personalized (5 min per email)
  - [ ] Research notes included (what did you learn about the org?)
  - [ ] Subject line chosen (which variation, A or B?)
  - [ ] CRM record created for each prospect
- [ ] Manager reviews and approves each email (check for typos, personalization quality)

### CRM Final Checks
- [ ] Workflows confirmed active (no test mode)
- [ ] Email tracking enabled (are opens/clicks being tracked?)
- [ ] Calendar integration confirmed (can sales rep see / book calls?)
- [ ] Slack integration confirmed (manager gets alerts for key events?)
- [ ] Dashboard tested (can you see current pipeline?)

### Sales Rep Readiness
- [ ] Sales Rep has access to:
  - [ ] CRM login (can view/edit contacts)
  - [ ] Email tracking (can personalize and send from CRM)
  - [ ] Calendar (can schedule discovery calls)
  - [ ] Sales playbook (printed or bookmarked)
- [ ] Sales Rep knows daily process:
  - [ ] 9 AM: Check for overnight replies (respond same day)
  - [ ] 10 AM: Send 10 cold emails (personalized, 5 min each)
  - [ ] 2 PM: Conduct discovery call (if any scheduled)
  - [ ] 4 PM: Update CRM with call notes, score prospects
  - [ ] 5 PM: Send follow-up email to call attendees (same day)

**Owner:** Sales Manager + Sales Rep

---

## DAY 12: SOFT LAUNCH

- [ ] Send first 10 cold emails (Sequence A, Email 1)
  - [ ] Emails are hand-personalized
  - [ ] Subject line tracked (which variant, A or B?)
  - [ ] Contacts marked as "cold email sent" in CRM
  - [ ] Outlook/Gmail shows "sent"
- [ ] Manager monitors next 2 hours:
  - [ ] Are emails getting delivered? (check bounces)
  - [ ] Are they being opened? (check tracking pixel)
- [ ] Adjust if needed:
  - [ ] If bounce: Verify email address
  - [ ] If spam: Check from address, confirm deliverability

**Owner:** Sales Manager + Sales Rep

---

## DAY 13-14: SOFT LAUNCH MONITORING + FULL LAUNCH PREP

### Soft Launch Monitoring (Day 13)
- [ ] 10 soft launch emails from Day 12:
  - [ ] Check open rate (target: 20%+)
  - [ ] Check if any bounced (fix or remove)
  - [ ] Check if any replied (if yes, manager takes over first discovery call for training)
- [ ] Adjust based on results:
  - [ ] If <15% open rate: Test new subject line on next batch
  - [ ] If any bounces: Validate email addresses
  - [ ] If reply comes in: Schedule discovery call, manager coaches sales rep

### Full Launch Prep (Day 14)
- [ ] Cold email list ready (100 prospects for Week 1 launch)
  - [ ] All personalization research completed
  - [ ] All email drafts approved by manager
  - [ ] Subject lines chosen (A/B test setup ready)
- [ ] CRM ready:
  - [ ] All automation active
  - [ ] All workflows tested
  - [ ] Dashboard live
  - [ ] Sales rep can access everything
- [ ] Reporting templates ready:
  - [ ] Weekly standup template (print or digital)
  - [ ] Weekly email report (template saved)
  - [ ] Monthly report (template saved)
- [ ] Manager briefing scheduled (Monday 9 AM):
  - [ ] Review Week 1 targets (50 emails, 2-3 discovery calls expected)
  - [ ] Confirm daily/weekly process
  - [ ] Q&A

**Owner:** Sales Manager + Sales Rep

## Red Flags (Stop & Reassess If...)

- [ ] Email open rate <15% for 2 weeks (quality or deliverability issue)
- [ ] No discovery calls scheduled by end of Week 1 (messaging or list issue)
- [ ] Discovery → consultation conversion <20% (qualification or script issue)
- [ ] 0 deals closed by end of Week 4 (sales process broken)
- [ ] Sales rep completely overwhelmed or burned out (reduce targets, add coaching)

**Action:** Diagnose the problem, don't just push harder. See contingency plans in sales-launch-implementation-plan.md.
